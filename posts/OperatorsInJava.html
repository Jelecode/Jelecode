Operators in the Java programming language

Operators are special lexical elements of the language, which enable the execution of the operations on values.

Operators in the Java language: 

    - arithmetic operators
    - assignment operators 
    - comparison operators
    - logical operators

Arithmetic operators

    + addition
    - subtraction
    * multiplication
    / division
    % modulo (remainder on division)

Example :

public class Main {

    public static void main (String [] args) {

        int x = 10;
        int y = 5;
        int modulo = 10%3;  //10 = 3*3+1
    
        System.out.println(x+y);
        System.out.println(x-y);
        System.out.println(x*y);
        System.out.println(x/y);
        System.out.println(modulo); 

    }

}

Exit : 

    15
    5
    50
    2
    1

Increment and decrement operators

    ++ increment operator (increment by one)
    -- decrement operator (decrement by one)

Example :

public class Main {

    public static void main (String [] args) {
                
        int x = 10;
        x++;

        System.out.println(x);
                
    }
                
}
                
Exit : 11

Assignment operators

    = assigns the value on the right to the variable on the left 

    += increments the value of the variable on the left by the value 
    on the right and assigns the resulting value to the variable on
    the left

    -= substracts the value of the variable on the left by the value 
    on the right and assigns the resulting value to the variable on
    the left

    *= multiplies the value of the variable on the left by the value 
    on the right and assigns the resulting value to the variable on
    the left

    /= devides the value of the variable on the left by the value 
    on the right and assigns the resulting value to the variable on
    the left

    %= calculates the remainder of deviding the value of the variable 
    on the left by the value on the right and assigns the resulting 
    remainder to the variable on the left

Example :

public class Main {

    public static void main (String [] args) {
                                
        int x = 10;
        x += 5;
                
        Sysstem.out.println(x);
                                
    }
                                
}
                                
Exit : 15

Comparison operators

    == equal to 
    != is not equal
    < it is less
    > it is bigger 
    >= greater than or qual to
    <= less then or equal to

Example :

public class Main {

    public static void main (String [] args) {
        
        int x = 1;
        int y = 2;

        System.out.println(x == y);
                                                
    }
                                                
}
                                                
Exit : false

Logical operators

    && - AND:
        (20>3 && 5>2)   true

Analysis of the AND operator :

true && true = true
false && true = false
true && false = false
false && false = false

    || - OR:
        (10<4 || 3>4)   true

Analysis of the OR operator :

true || true = true
false || true = true
true || false = true
false || false = false

    !  - NOT:
        !(6==6)     false

Analysis of the NOT operator :

!true = false
!false = true

Ternary operator

    variable = (condition) ? value1 : value2;

Meaning og the giving syntax : If the condition is met, assign the value1 to the variable, and if the condition is not met, assign the value2 to the variable.

Example :

public class Main {

    public static void main (String [] args) {
        
        int age = 14;

        String message = (age < 18) ? "You are underage!" : "Welcome to the site" ;

        System.out.println(message);
                                                                
    }
                                                                
}
                                                                
Exit : You are underage!

Concatenation of strings

Arithmetic operator for addition ( + ) in the Java language has another function - addition or concatenation of strings.

Example :

public class Main {

    public static void main (String [] args) {

        String value1 = "Hello"
        String value2 = "World"

        System.out.println(value1 + value2);
                                                                               
    }
                                                                                
} 

Exit : HelloWorld

You can also add text to an existing string

public class Main {

    public static void main (String [] args) {
                
        String value1 = "Hello"
                
        System.out.println(value1 + " World!");
                                                                                               
    }
                                                                                                
} 
                
Exit : Hello World!
