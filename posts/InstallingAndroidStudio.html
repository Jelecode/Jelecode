Download and install the Android Studio development enviroment

To establish an enviroment for the successful creation of Android applications, it is sufficient to download the following two components: 

    - JDK
    - Android Studio

Since version 2.2 of the Android Studio development enviroment, OpenJDK is an integral part of it, so for developing applications it is quite sufficient to install only the development enviroment Android Studio.

However, in certain situations it isn't bad to have an independent JDK.

Check this post for detailed steps on how to install OpenJDK on your computer:

Setting up Java Development Kit (JDK)

Android Studio

To download the installation file of the Android Studio development enviroment, visit the website at the following address:

https://developer.android.com/studio/index.html

When installing by clicking on the "DOWNLOAD ANDROID STUDIO" button, a window with licensing conditions that must be accepted before starting the download is displayed.

After downloading the installation file of Android Studio development enviroment, it is necessary to start the installation by double-clicking on the downloaded file.

On the first page of the installation window, you only need to click on the "Next" button.

On the second page are the components that will be installed. The development enviroment, Android SDK, and the virtual device necessary for the testing created applications. It is necessary to leave all listed components checked and click on "Next".

The next page brings a text of the licence, which must be accepted by selecting "I Agree".

On the next page there are options regarding the paths where the development enviroment and the Android SDK will be installed. It is the best to choose the same hard-disk where the JDK is installed.

This page provides an overview of the Stert menu folder, where you will find the icon for launching Android Studio development enviroment and "Install" button.

After installation, checking "Start Android Studio" and clicking "Finish" the development enviroment will be started.

Starting the development environment for the first time displays a welcome screen where you can make some additional settings.

Clicking the "Finish" button the necessary components will be downloaded and the setup of the development enviroment will be completed.

After completing the necessary settings, the inital screen of the development enviroment opens.
