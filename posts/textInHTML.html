Text in HTML

The basis of every HTML document is text.
However, text within an HTML document can have different contexts, just like within any other text document, where different text sections can be layered.

Thus, under the term text can be mentioned: titles, paragraphs, transitions to a new line, empty spaces, vertical lines, as well as various other forms of text that acquire a special meaning through special formatting.

We will cover the following topics:

- headings

- paragraphs

- text formatting

- special characters

Headings

Each document begins with a title.
Different parts of the deocument can be identified by subheadings of different levels. The HTML language knows 6 levels of headings that are created using the following elements: 


<h1>
<h2>
<h3>
<h4>
<h5>
<h6>

Examples of different HTML headings:


<h1>Lorem ipsum dolor sit amet</h1>

<h2>Lorem ipsum dolor sit amet</h2>

<h3>Lorem ipsum dolor sit amet</h3>

<h4>Lorem ipsum dolor sit amet</h4>

<h5>Lorem ipsum dolor sit amet</h5>

<h6>Lorem ipsum dolor sit amet</h6>

Paragraphs

In an HTML document, paragraphs are created by placing them between tags:


<p></p>

HTML elements for text formatting

The HTML language knows special elements for defining text using different formatting.
Those special elements are: 


<b> - bold text 

<strong> - important text 

<i> - italic text 

<em> - highlighted text 

<mark> - marked text 

<small> - small text 

<del> - deleted text 

<ins> - inserted text 

<sub> - subscript 

<sup> - superscript

New line

If it is necessary to independently influence where the text line break will occur, it is possible to use tag: 


<br>

This element doesn not have its own content and is one of self-closing elements, since it does not have a closing tag.

Hr

<hr>

Hr is an element that enables the thematic transition of the text.
However, this element got its name from the term horizontal rule, which is a line that can be used to separate two sections of text.

Spaces and lines in the text

It is very important to note a few things about spaces and new lines in the text:

- HTML does not handle multiple spaces and newlines

- Multiple spaces are always treated as a single space

- Newlines breaks are ignored.

If it is necessary to move to a new line, use element br.

If there is ever a need to preserve multiple spaces, ad well as transitions to a new line without using the br element, it is possible to us the pre element.
The entire content inside the pre element retains all spaces and newlines.

Special characters

Certain characters in the HTML language have a special meaning.

The ideal representatives of such characters are certainly the < and > characters, which are used to build HTML tags.
When an HTML tag is written, the browser does not display it on the page in an original form, but interprets it and its possible content in a specific way.

If we want these characters to be displayed inside the browser in their origilan form, we must use replacement characters.

Replacement characters or HTML entities are useful when we need to solve the problem of reserved characters, but also for writing special symbols.

Some of the most useful HTML entities are:


&nbsp;    non breaking space

&lt;      < 

&gt;      >

&amp;     &
