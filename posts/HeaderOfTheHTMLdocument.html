Header of the HTML document

Each HTML document, in addition to the elements that are visually presented on the page, also has its header, with elements that are mostly not visible on the page, but they play a very important role in its functioning.

The root element of every HTML document is the html element. Is starts with the opening tag

<html> and ends with the closing tag </html>.

The most important features of the html element :

- tells the browser that it is an HTML document

- represents the root element of the HTML document

- includes all other elements on the page.

The html element allows defining two attributes :


manifest - defines the cache manifest address

xmnls - defines an XML namespace

The two direct descendants of the html element are the elements : 


head - <head></head>

body - <body></body>

The body element contains the body of the document and it is directly visible on the page, the head element represents the document header.

Head element (header of the document)

The head element is the root element for all header data of an HTML document.

This element is the parent element for the following HTML elements : 


<title></title>

<style></style>

<base>

<link>

<meta>

<script></script>

<noscript></noscript>

- title element

This is a mandatory element of every HTML document. Defines the name of the document. One document can have only one title element.

- style element

This element is used to define the CSS styling of an HTML document. This styling determines how the elements will be displayed on the browser page. It is possible to have more than one style element in one document.

- base element

This element defines the base address for all relative addresses within the HTML document. It is allowed to define only one base address per page, and it must be defined inside the head element.

- link element

This element defines a link to an external document or resource. It is most often used to include an external styling file. It can only be found within the head section but can appear any number of times.

- meta element

This element is used to define metadata about an HTML document. Metadata are not displayed on the page, but are therefore parsed by browser, search engines and other web services.

- script element

This element is used to define client-side scripts. It can contain code directly, or refer to an external client script file.

- noscript element

This element is used to define alternative content that will be displayed to users whose browser have script execution disabled or not supported.
