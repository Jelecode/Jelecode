Switch

Switch is a flow control statement within the Java programming language. This statement performs flow control in a slightly different way than the if...else statement.

The syntax of the Switch statement : 


switch (expression) {

    case label_1:
        statements_1;
        break;

    case label_2:
        statements_2;
        break;

            ... 

    default:
        statements_default;
        break;

}

The switch statement compares the expression value with the case clause values. In the shown example, the value of the case clasues are label_1 , label_2 , etc.

When the matching of the expression value with some of the case clasues is determined, the execution of the program is redirected to that block. In case no matches are found, the default block is executed.

Example of using a switch statement

A practical example of using the switch statement might look like this :


int x = 1;

switch (x) {

    case 0:
        System.out.println("zero");
        break;

    case 1:
        System.out.println("one");
        break;

    default:
        System.out.println("unknown value");
        break;

}

In the shown example, a switch command was created which, depending on the value of the variable x , transmits a certain message.

When the value of variable x is equal to 0, zero is printed on the output.
When the value of the variable x is equal to 1, one is printed on the output.
Finally, if the value of the variable x is anything other than the numbers 0 and 1, the text unknown value is printed on the output.

Within the switch statement, the equality of the contol value (which is x in the example) with the values of individual case blocks (values after the keyword case) is determined.
In other words, switch allows creation of logical conditions in which equality is checked, but not other logical types (greater than, less than, unequally, ...).

The control value of the switch command can be of the following type: char, byte, short, int, Character, Byte, Short, Integer, String, enum .

The default code block is, by convention, placed at the end of the switch statement, but this does not have to be rule. This block of code can be placed anywhere within a switch statement.

Each of the switch cases must end with a break statement

Each of the cases inside the switch statement has a break keyword at the end.
The role of the break keyword is to finish the execution of the code inside the switch statement.
When the execution environment encounters this statement, execution of the switch statement ends.
If this command is not specified at the end of the cases within the switch construction, the switch will not function correctly.

Improved switch

The previous examples illustrated the use of the standard switch construction, which exists in most other programming languages.
However, the Java language additionally improves the switch command, so it is possible to use it in a slightly different, more compact way. Such a switch is called an improved switch.

Example of the impoved switch in Java :


int x = 1;

switch (x) {
    case 0 -> System.out.println("zero");
    case 1 -> System.out.println("one");
    default -> System.out.println("unknown value");
}

Even at first glance, it is clear that the code just shown is much more compact than the one from the previous example.
Also, the improved switch frees us from the need to define a break statement at the end of each case, which is a huge advantage of the Java language.
