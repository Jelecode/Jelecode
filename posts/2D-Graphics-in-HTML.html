2D Graphics in HTML

For displaying graphics within an HTML document, HTML5 has two technologies : 

    - SVG 
    - Canvas

SVG

Scalable Vector Graphics is an XML based descriptive language used to create 2D graphics. HTML element used to draw SVG graphics is :

<svg></svg> svg- presents a container for graphic elements.

The following example illustrates how a rectangle can be drawn using SVG 


<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
        <meta charset="utf-8"/>
    </head>
    <body>

        <svg version="1.1"
             baseProfile="full"
             width="300" height="200"
             xmlns="https://www.w3.org/2000/svg">

             <rect x="10" y="10" 
             width="200" height="100" 
             fill="#4D4D4D"
             stroke="#4F95FF" 
             stroke-width="3" />

        </svg>
    </body>
</html>

Since the SVG language is an XML derivative, the XML namespaces is specified within the svg tag using the xmlns attribute.

Within the svg element, the height and width of the window in witch the 2D graphics will be displayed are defined.

The SVG language itself defines elements for drawing various graphic forms. In the example, the rect element is used, witch draws a rectangle.

Attributes on the rect element :

    x - x coordinate of the upper left point
    y - y coordinate of the upper left point
    stroke-width - frame line tickness
    fill - fill color
    stroke - frame color

SVG recognizes the following basics graphic forms :

    - rectangle (rect)
    - circle (circle)
    - ellipse (ellipse)
    - line (line)
    - polyline (polyline)
    - polygon (polygon)
    - path (path)

Attributes

RECTANGLE (rect) :

    x - x coordinate of the upper left point
    y - y coordinate of the upper left point
    width - the width of the rectangle
    height - the height of the rectangle
    rx - x radius of the edge of the rectangle
    ry - y radius of the edge of the rectangle

CIRCLE (circle) : 

    r - radius of the circle
    cx - x coordinate of the center of circle
    cy - y coordinate of the center of circle

ELLIPSE (ellipse) : 

    rx - x radius of the ellipse
    ry - y radius of the ellipse
    cx - x coordinate of the center of ellipse
    cy - y coordinate of the center of ellipse

LINE (line) : 

    x1 - x coordinate of the starting point
    y1 - y coordinate of the starting point
    x2 - x coordinate of the end point
    y2 - y coordinate of the end point

POLYLINE (polyline) : 

    points - array of comma-separated points,
            defined by x and y coordinates

Example : 


<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
        <meta charset="utf-8"/>
    </head>
    <body>
                    
        <svg version="1.1"
            baseProfile="full"
            width="300" height="200"
            xmlns="https://www.w3.org/2000/svg">
                    
                <polyline 
                  points="30 55,60 25,100 65,140 20"
                  fill="none"
                  stroke="#4F95FF" 
                  stroke-width="3" />
                    
        </svg>
    </body>
</html>

POLYGON (polygon) : 

    points - array of comma-separated points,
            defined by x and y coordinates

Example : 


    <polygon 
        points="20 65,60 5,100 65"
        stroke-width="3"
        stroke="#4F95FF"
        fill="#4D4D4D" />

PATH (path) : 

    d - array of comma-separated points,
            defined by x and y coordinates

Example : 


    <path 
        d="M10 10 C 20 20,40 20,50 10"
        stroke="#4F95FF"
        fill="none" />

Canvas

The Canvas element is used to programmatically draw graphics using the JavaScript language. The Canvas element is most often used for drawing animated graphics.

Canvas is created using element :

<canvas></canvas>

Example : 


<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
        <meta charset="utf-8"/>
    </head>
    <body>
        <canvas id="myCanvas" width="200" height="200"></canvas>

        <script>
            var c = document.getElementById("myCanvas");
            var ctx = c.getContext("2d");
            ctx.beginPath();
            ctx.fillStyle="red";
            ctx.arc(95, 50, 40, 0, 2*Math.PI);
            ctx.fill();
        </script>
    </body>
</html>
