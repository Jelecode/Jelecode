Intraduction to HTML

One HTML document can contain text, presented in various forms, but also images, videos, sounds, graphs, links, 2D graphics and animation. In order to porperly define desired content of an HTML document, it is necessary to know the rules that dictate the HTML language.

HTML is used to define the structure of an HTML document. To perform such work, HTML has a large number of different tags, which are used to define sections of the HTML document.

A simple HTML page can have the following structure :


<!DOCTYPE html>
<html>
<head>       
    <title>This is title</title>         
</head>
                
<body>
    <h1>This is a heading</h1>
    <p>This is first paragraph</p>
    <p>This is second paragraph</p>
</body>
</html>

If this code is saved in an text document with the extension .html , and opened in a web browser, it will be displayed as in the picture.

By analyzing the image, it can be concluded that the tag title refers to the name of the document, tag h1 to the heading, and tag p to the paragraphs in the document.

Finally, it can be concluded that the elements, tags and attributes are the basic building blocks of an HTML document.

Elements and tags

The HTML document has 3 different types of tags :

- opening tags

- closing tags

- self-closing tags

An example of an HTML tag is : 


<h1>

Also one HTML tag is :


</h1>

A certain difference can be observed between the two shown tags. The second tag contains a slash character in front of its name, which means that it is a closing tag, the first tag without a slash character is an opening tag.

In addition to opening and closing tags, HTML has a special type of tags called self-closing tags. These are tags that start and end in the same time. An example of a self-closing tag : 


<br />

A self-closing tag has a slash character at the end of the tag name, which must be separated from the tag name by a blank space.

The described HTML tags are used to build an HTML element. The element starts with an opening tag and ends with a closing tag. For example, a heading is an HTML element : 


<h1>This is heading</h1>

Element attributes

Every HTML element can contain attributes. Attributes provide additional informations about HTML elements, and they are always defined on the element's opening tag. Attributes usually occur as a key-value pairs, where the name of the attribute is key. 


<h1 id="first_heading">This is heading</h1>

Global HTML attributes

Different HTML elements have a different set of attributes that can be defined on them. However, certain HTML attributes can appear on any element.

Global HTML attributes are :

accesskey  

class

contenteditable

contextmenu

data-* 

dir 

draggable 

dropzone

hidden 

id 

lang 

spellcheck 

style 

tabindex 

title 

translate

accesskey - defines the keyboard key that adds focus to the element.

class - defines one or more element class names.

contenteditable - defines whether the content of the element is editable or not.

contextmenu - defines a context menu that is obitained by right-clicking on element.

data-* - defines custom application data.

dir - defines the direction of writing text inside the element.

draggable - defines whether the element can be dragged around the page.

dropzone - defines whether the data will be copied, moved or linked when released after being dragged.

hidden - defines an element as invisible.

id - defines a unique element identifier.

lang - defines the content language of the element.

spellcheck - defines whether the content of the element will be grammatically checked.

style - defines the inline CSS styling of the element.

tabindex - defines the order of elements on the page when using the tab key to move from element to element.

title - defines additional informations about the element.

translate - defines whether the content of the element will be translated.

HTML document structure

HTML pages generally contain certain basic elements. These are the following elements : 


- <html>
- <head>
- <body>

The basic structure that every HTML element has is the following :


<!DOCTYPE html>
<html>
<head>       
             
</head>            
<body>
    
</body>
</html>

An HTML element begins with a DOCTYPE declaration. This declaration tells the browser which version af HTML is used in document.

After the declaration there is an html element. It is an element that surrounds all other elements within the document and indicated that it is HTML code.

Inside the html element, the first element in the basic structure is head. Elements that are not directly visible on the page are placed inside the head element. For example meta tags, links to external CSS and other files, ...

The body tag follows immediately after the head tag, and contains everything that is directly visible on the page.
