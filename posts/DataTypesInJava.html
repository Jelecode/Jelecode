Data types in the Java programming language

Data types allow different types of values to be represented within a program.

The Java Virtual Machine uses data types to determine what type the input data is.

Data types in Java are :

- simple or primitive types

- complex or object types

Primitive types

The Java programming language has 8 built-in primitive data types :


        byte

8-bit integer value (from -128 to 127)


        short

integer value of the reduced range (from -32 768 to 32 767)


        int

standard range integer value (from -2 147 483 648 to 2 147 483 647)


        long

integer value of the extended range (from -9 223 372 036 854 775 808L to 9 223 372 036 854 775 807L)


        float

decimal value (+- 3.40282347 E+38F)


        double

decimal value double precision (+- 1.79769313486231570 E+308)


        char

character (0-65536)


        boolean

logical value (true/false)

Accordingly, we devide the primitive types into :

- integer types : byte, short, int, long

- decimal values types : float, double

- characters types: char

- logical types : boolean

Complex/object types

Complex data types got their name based on the property that allows them to combine a large numbers of values, but also operations, behaviors,..

Each data primitive type has its own wrapper, which is a complex type : 


byte - Byte

short - Short 

int - Integer 

long - Long 

float - Float 

double - Double 

boolean - Boolean 

char - Character

We will best familiarize ourselves with the differences between primitive and complex data types using the example of primitive type wrappers : 

Integer x = 5;

At first glance, the difference is only the keyword, but complex types are not limited to value only, but also possess many other behaviors.

Text data type

Text values are represented by the String type in the Java programming language.

The String type is a complex type, that's why its name starts with the first capital letter, but it can be used in Java like primitive types.

The value of the String type is written between double quotes, unlike char whose value is enclosed in single quotes.

Backslash is used in Strings so that the character after backslash is treated differently by the Java compiler.

Using backslash in Strings :


\t      tab space 

\b      backspace

\n      new line

\r      return to the beginning of the line

\'      single quote

\"      double quotes

\\      slash
