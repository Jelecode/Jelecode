If, else if, else

Controlling the flow of execution in Java can be achived by branching (conditional execution) and looping (repetition).

Branching

Branching allows ceratin commands to be executed only if a condition is met. This is why branching is closely related to logical data types and logical operators.

Java makes it possible to create several possible branches of execution, and the selection of the branch of execution itself depends on some control logic value.

Java recognized two types of conditional statements :

- if ... else;

- switch.

If statement

The basic branching statement in Java is the if statement. This command exists in practically all programming languages, and its purpose is reflected in the name itself.

The word if, by itself, does not make much sense in retoric, and the same in programming.
In addition to this keyword, it is necessary to set some condition on which the outcome of this command will depend.
So the general syntax of the if statement looks like this : 


if (condition) {
    //code to execute if condition is met.
}

To form an if statement, the keyword if is used, followed by a condition in brackets. If the condition is met, the code block of the if statement is executed.

An example that illustrates the use of the if statement :


int age = 17;

if(age < 18) {
    System.out.println("Underage");
}

At the beginning, a variable with the name age an the value 17 is declared and initialized. After this line, the if command follows. The condition is defined to check whether the value of the age variable is less then 18.
If this condition is met, the block of code between curly brackets is executed, which in the example has only one command.

If the value of the variable age was greater than or equal to the 18, the block of code from the example would not be executed.

Blocks of the conditional statements

In the example just shown, you could see that the if statements consists of a condition and a block.
A block is formed like any other block in the Java language, using curly brackets. 


if(age < 18) {
    System.out.println("Underage");
}

However, it is important to know that the brackets for forming a block can be omitted if the block has only one command : 


if(age < 18) 
    System.out.println("Underage");

It is important to remember that in this case an if conditional block can have only one statement. Any command after that will not be part of the if block.

if...else statement

If it is necessary to execute a specific block of code when the if condition is not met, the keyword else is used, so if becomes if...else .

if...else is a flow control command that allows defining the logic that will be executed if the condition is met, but also the logic that will be executed if the condition is not met.
The syntax of the if...else state is as follows :


if(condition) {
    statement_1;
} else {
    statement_2;
}

After the keyword if , the condition is specified in brackets.
If this condition is met, the first statement is executed (statement_1), otherwise the second statement is executed (statement_2).

Depending on whether the condition is met, one of these two code blocks is executed. It is important to understand that one block of code will need to be executed. Which block it will be depends on the condition.

An example of using if...else statement :


if (age < 18) {
    System.out.println("Underage");
} else {
    System.out.println("Adult");
}

In the example, the condition age<18 is defined, if the age is less than 18, the code block that prints "Underage" will be executed, otherwise the code block that prints "Adult" will be executed.

An if...else statement can have only one if block and only one else block. The condition is always defined only on the if block.

if...else if... else statement

In the previous lines, it was shown that the if...else statement can have only one condition and that is on the if block.
If it is necessary to define additional code blocks with their own conditions, if...else if...else statement is used.

else...if enables additional conditions to be defined when creating an if...else statement, so if...else statement becomes if...else if...else statement.

The syntax of such commands is as follows :


if(condition_1) {
    statement_1;
} else if(condition_2) {
    statement_2;
} else if(condition_3) {
    statement_3;
} else {
    statement_4;
}

In this example, in addition to the if and else blocks, there are also two else if blocks.
If the first condition is met (condition_1), statement_1 will be executed. If the second condition is met (condition_2), statement_2 will be executed. If the condition_n is met, statement_n will be executed.

It is important to understand that the number of else if blocks can be arbitrary.

If none of the conditions are met, the command in the else block is executed.

An example of using the if...else if...else command is as follows :


int age = 32;

if(age < 18) {
    System.out.println("Underage");
} else if(age < 30) {
    System.out.println("18 - 30");
} else if(age < 40) {
    System.out.println("30 - 40");
} else if(age < 50) {
    System.out.println("40 - 50");
} else if(age < 60) {
    System.out.println("50 - 60");
} else {
    System.out.println("Older than 60");
}


Exit :
30 - 40

The first command declares and initializes the age variable with the value 32.

if...else if...else command has 5 conditions. If the value of variable age is less than 18, the first code block is executed. Since this is not the case, the second condition (age<30), is examined, since it is also not true, the third condition (age<40), is examined.
The third condition is met, because 32 is less than 40, so the execution of the program enteres this else if block and the message "30 - 40" is printed.

It is important to point out another very important feature of the shown construction. The fourth block has the condition age<50 and the fifth block has the condition age<60.
If these conditions are analyzed, it can be concluded that they are also met, because 32 is less then 50 and 32 is less then 60.
However, within the statement if...else if...else , only one block of code is always executed, and that is the first one whose condition turns out to be true.

Ternary operator

By using the ternary operator, it is possible to assign a value to a variable based on some condition :

String message = (age<18) ? "You are underage" : "You are adult";

The condition is age<18 , and the potential values that could be assigned to the message variable are "You are underage" and "You are adult".

If the condition is met, "You are underage" is assigned to the variable message , otherwise "You are adult" is assigned to the variable.

The ternary operator can always be transformed into an if...else statement.
Thus, the shown logic can be transformed as follows : 


String message = "";
int age = 19;

if(age < 18) {
    message = "You are underage";
} else {
    message = "You are adult";
}

System.out.println(message);


Exit : 
You are adult
